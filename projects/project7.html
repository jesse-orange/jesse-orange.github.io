Sound of Drawing

Project Content

This is a creative project that aims to design a program to verify how different images can be transformed into distinct musical melodies, thereby further interpreting the relationship between images and music from a philosophical perspective.

Program Development

To ensure the developed program is installation-free and compatible with multiple operating systems, web technologies such as HTML, CSS, and JavaScript were primarily used in its development.

Program Functions

  • Draw images online or upload images
  • Analyze images and transform them into music
  • Play back the transformed music
  • Generate relevant analysis reports

Philosophical Interpretation

This project examines the philosophical relationship between creation and perception, questioning traditional boundaries between sensory experiences. It explores how artistic expression transcends medium-specific limitations.
